# ramne

All file names randomly rename in the directory.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ramne'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ramne

## How to use?

    $ bin/ramne ~/target_directory/

Or

    $ bin/ramne ~/target_directory/ 8
